# Image filter

Apply a sepia or black/white filter to an image using JavaScript canvas.

1) Install dependencies:
```sh
npm install
```

2) Start a server:
```sh
python -m SimpleHTTPServer
```

3) Run npm watch task:
```sh
npm run watch-js
```

4) Point browser to:
```sh
http://localhost:8000/
```
